Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de apresentar meu portfólio pessoal como estudante de programação. Ele reúne informações sobre quem sou, minhas habilidades técnicas e alguns dos projetos que já desenvolvi ou estou em andamento.

O portfólio foi criado utilizando HTML, CSS e uma estrutura responsiva com ajuda do Figma para o design visual. Também estão incluídos: uma introdução pessoal, minhas principais competências (como HTML, CSS, Figma e Scratch), e uma galeria de projetos.

Esse projeto foi uma atividade proposta na disciplina de Programação, no 2º ano do Ensino Médio, e tem como finalidade mostrar meu desenvolvimento como estudante, além de servir como vitrine para oportunidades futuras em tecnologia e design.

Ele representa um passo importante no meu aprendizado e crescimento na área de programação e design digital.
